# Images Rucher360

Ces visuels generes par IA servent uniquement d'habillage applicatif pour le
shell Rucher360. Ils ne contiennent pas de donnees personnelles, de secrets, de
logos tiers, de texte utilisateur ou d'information de production.

## Assets

- `hero-cockpit-apiary.png` - hero cockpit, 1672 x 941.
- `module-apiaries-banner.png` - page ruchers, 1672 x 941.
- `module-hive-card.png` - page ruches, 1448 x 1086.
- `module-visits-field.png` - page visites, 1448 x 1086.
- `module-equipment-inventory.png` - page materiel, 1448 x 1086.
- `module-tasks-checklist.png` - page taches, 1448 x 1086.
- `module-transhumance-banner.png` - page transhumance, 1672 x 941.
- `module-health-observation.png` - page sanitaire et connaissance, 1448 x 1086.
- `module-hive-square.png` - vignette secondaire ruche, 1254 x 1254.

## Regles d'utilisation

- Utiliser les images comme decoration ou illustration statique.
- Ne pas les presenter comme des donnees terrain reelles.
- Ne pas ajouter d'image contenant une personne identifiable, un secret, une
  adresse precise ou un contenu personnel.
- Optimiser ou remplacer les fichiers dans un lot dedie si le poids devient un
  probleme de performance.
